## מה זה עושה?

1. **מתמלל** - הופך הקלטת אודיו לטקסט (עם Whisper, תומך עברית מעולה)
2. **מסכם** - מייצר סיכום חכם עם משימות לביצוע (עם מודל AI מקומי)
3. **שומר** - שומר הכל בקובץ Markdown מסודר

**הכל רץ מקומית על ה-Mac M4 שלך. אף בייט לא יוצא מהמחשב.**

## התקנה (5 דקות)

### אפשרות א': התקנה אוטומטית
```bash
chmod +x install.sh
./install.sh
```

### אפשרות ב': התקנה ידנית

**שלב 1: התקנת Ollama (מנוע ה-AI)**
```bash
# הורד מהאתר:
# https://ollama.com/download
# או עם Homebrew:
brew install ollama
```

**שלב 2: הפעלת Ollama והורדת מודל**
```bash
# הפעל את Ollama (צריך לרוץ ברקע)
ollama serve

# בטרמינל חדש - הורד מודל סיכום (1.7GB, פעם אחת)
ollama pull qwen3:1.7b
```

**שלב 3: התקנת חבילות Python**
```bash
# צור סביבה וירטואלית (מומלץ)
python3 -m venv ~/.localscribe_env
source ~/.localscribe_env/bin/activate

# התקן mlx-whisper (תמלול מהיר על Apple Silicon)
pip install mlx-whisper
```

**שלב 4: התקנת ffmpeg (לעיבוד אודיו)**
```bash
brew install ffmpeg
```

## שימוש

### 🧪 בדיקה מהירה (בלי הקלטה)
```bash

```
זה ירוץ עם טקסט לדוגמה כדי לוודא שהסיכום עובד.

```bash

```
תומך בפורמטים: mp3, wav, m4a, mp4, webm, ogg

## דוגמה לתוצאה

נניח שהקלטת פגישת צוות של 30 דקות. התוצאה תיראה ככה:

```markdown
# סיכום פגישה
**תאריך:** 03/05/2026 14:30

## סיכום
פגישת צוות שדנה בהשקת המוצר החדש ב-25 למאי, חריגה תקציבית 
והצורך בגיוס מפתח נוסף.

## משימות לביצוע
1. דנה - לסיים עיצוב דף נחיתה עד יום שלישי
2. מיכל - לשלוח טקסטים סופיים מחר בבוקר
3. אורי - לעדכן קמפיינים (להעביר מפייסבוק לגוגל) עד חמישי
4. מיכל - לשלוח קורות חיים של מועמדים היום

## החלטות
- השקה ב-25 למאי כמתוכנן
- קיצוץ תקציב פרסום בפייסבוק, העברה לגוגל
- גיוס מפתח פולסטאק - ראיונות שבוע הבא
```

## שאלות נפוצות

**ש: כמה מקום זה תופס?**
ת: ~5GB סה"כ (3GB מודל תמלול + 1.7GB מודל סיכום). פעם אחת.

**ש: זה עובד בלי אינטרנט?**
ת: כן! אחרי ההתקנה הראשונית (שדורשת אינטרנט להורדת המודלים), הכל עובד אופליין.

**ש: זה עובד בעברית?**
ת: כן! Whisper Large V3 תומך בעברית ברמה גבוהה מאוד. הסיכום גם בעברית.

**ש: אפשר להשתמש בזה לשיחות טלפון?**
ת: כן, אם תקליט את השיחה (למשל עם אפליקציית הקלטה) ותעביר את הקובץ לסקריפט.

**ש: מה ההבדל בין זה ל-Otter.ai?**
ת: Otter שולח הכל לענן (לא פרטי) ועולה $100+ בשנה. אצלנו הכל מקומי וחינם.
